# Marketing Pro Workflow

数字营销人员、社交媒体经理专用的Claude Code工作流。

## 适合人群

- 数字营销人员
- 社交媒体经理
- 企业营销团队
- 品牌运营专员

## 已安装Skills

| Skill | 功能 | 触发示例 |
|-------|------|---------|
| seo-optimizer | SEO内容优化 | "优化这篇文章的SEO" |
| linkedin-post-optimizer | LinkedIn帖子优化 | "优化LinkedIn帖子" |
| social-repurposer | 跨平台内容转换 | "转换为各平台格式" |
| landing-page-copywriter | 落地页文案 | "写落地页文案" |
| email-template-generator | 邮件营销模板 | "生成邮件营销序列" |

## 快速开始

```bash
# 克隆并进入目录
cd marketing-pro-workflow

# 启动Claude Code，skills自动激活
```

## 使用示例

### 营销活动流程

```
1. "为 [产品] 写一个落地页文案"
2. "用 seo-optimizer 优化内容"
3. "生成邮件营销序列（3封）"
4. "转换为各社交媒体平台的帖子"
5. "优化LinkedIn版本"
```

### SEO内容优化

```
"用 seo-optimizer 优化这篇文章，目标关键词是 [keywords]"
```

## 多AI工具支持

本workflow支持：
- Claude Code (`.claude/skills/`)
- Codex (`.codex/skills/`)
- Cursor (`.cursor/skills/`)
- OpenCode (`.opencode/skill/`)
