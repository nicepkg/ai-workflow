# Marketing Pro Workflow - AI Instructions

AI execution guide for marketing professional workflow.

## Available Skills

| Skill | Trigger |
|-------|---------|
| seo-optimizer | SEO, optimize keywords, search ranking |
| content-optimizer | keyword density, meta tags, heading structure, readability |
| serp-analysis | SERP analysis, search results, competitor analysis |
| keyword-cluster-builder | keyword research, keyword expansion, topic clusters |
| analytics-interpretation | GA4, GSC, analytics, metrics, traffic analysis |
| content-brief | content brief, content plan, SEO brief |
| linkedin-post-optimizer | LinkedIn, professional post optimization |
| social-repurposer | social media conversion, multi-platform |
| landing-page-copywriter | landing page, conversion copy |
| email-template-generator | email marketing, email sequence, newsletter |

## Recommended Workflow

### SEO Content Strategy

1. **Keyword Research** - Use keyword-cluster-builder to expand and cluster keywords
2. **SERP Analysis** - Use serp-analysis to understand search intent and competition
3. **Content Planning** - Use content-brief to create detailed content requirements
4. **Content Creation** - Write content following the brief
5. **Optimization** - Use content-optimizer to verify SEO requirements

### Marketing Campaign Flow

1. **Copywriting** - Use landing-page-copywriter to create core copy
2. **SEO Optimization** - Use seo-optimizer to optimize content
3. **Email Sequence** - Use email-template-generator to create emails
4. **Social Distribution** - Use social-repurposer to convert formats
5. **LinkedIn Optimization** - Use linkedin-post-optimizer for professional optimization

### Analytics Review

1. **Data Collection** - Gather GA4/GSC data
2. **Interpretation** - Use analytics-interpretation to understand metrics
3. **Action Planning** - Generate recommendations based on data

## Skill Combinations

- **SEO Campaign**: keyword-cluster-builder → serp-analysis → content-brief → content-optimizer
- **Product Launch**: landing-page-copywriter → email-template-generator → social-repurposer
- **Content Marketing**: content-brief → seo-optimizer → social-repurposer → linkedin-post-optimizer
- **Performance Review**: analytics-interpretation → content-optimizer (for underperforming content)

## Output Standards

- Landing Page: AIDA framework, clear CTA
- Email Sequence: 3-5 emails, progressive engagement
- LinkedIn: Professional tone, under 1300 characters
- SEO: Include meta tags, keyword suggestions
- Content Brief: E-E-A-T requirements, competitor analysis
- SERP Analysis: Intent classification, feature opportunities, competitor matrix
- Keyword Clusters: Grouped by topic and intent, funnel stage mapping
