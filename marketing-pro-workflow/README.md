# Marketing Pro Workflow

Claude Code workflow for digital marketers and social media managers.

[中文文档](./README_cn.md)

## Target Users

- Digital Marketers
- Social Media Managers
- Marketing Teams
- Brand Operators

## Installed Skills

| Skill | Function | Trigger Example |
|-------|----------|-----------------|
| seo-optimizer | SEO content optimization | "Optimize this article for SEO" |
| linkedin-post-optimizer | LinkedIn post optimization | "Optimize LinkedIn post" |
| social-repurposer | Cross-platform content conversion | "Convert to all platform formats" |
| landing-page-copywriter | Landing page copy | "Write landing page copy" |
| email-template-generator | Email marketing templates | "Generate email marketing sequence" |

## Quick Start

```bash
# Navigate to directory
cd marketing-pro-workflow

# Launch Claude Code - skills activate automatically
```

## Usage Examples

### Marketing Campaign Flow

```
1. "Write landing page copy for [product]"
2. "Use seo-optimizer to optimize the content"
3. "Generate email marketing sequence (3 emails)"
4. "Convert to social media posts for each platform"
5. "Optimize LinkedIn version"
```

### SEO Content Optimization

```
"Use seo-optimizer to optimize this article, target keywords are [keywords]"
```

## Multi-AI Tool Support

This workflow supports:
- Claude Code (`.claude/skills/`)
- Codex (`.codex/skills/`)
- Cursor (`.cursor/skills/`)
- OpenCode (`.opencode/skill/`)
