## ⚡ 快速安装

```bash

npx add-skill nicepkg/ai-workflow/video-creator-workflow

# 或安装单个技能
npx add-skill nicepkg/ai-workflow/video-creator-workflow --skill video-script-writer
```

## 📦 包含技能 (15个)

### 选题与研究
| 技能 | 功能描述 |
|:-----|:---------|
| `content-trend-researcher` | 跨 10+ 平台热点追踪 |
| `youtube-transcript` | 提取视频字幕用于研究 |
| `video-downloader` | 下载视频作为参考素材 |

### 脚本与钩子
| 技能 | 功能描述 |
|:-----|:---------|
| `video-script-writer` | 撰写完整视频脚本含结构 |
| `video-hook-generator` | 生成病毒式开场钩子（前3秒） |

### 标题与封面
| 技能 | 功能描述 |
|:-----|:---------|
| `video-title-optimizer` | 优化标题以提高点击率和 SEO |
| `thumbnail-concept-generator` | 创建缩略图概念和设计简报 |
| `canvas-design` | AI 视觉设计生成缩略图 |

### SEO 与分发
| 技能 | 功能描述 |
|:-----|:---------|
| `youtube-seo-optimizer` | 优化描述、标签、章节 |
| `short-form-converter` | 长视频转短视频/TikTok |
| `social-repurposer` | 跨平台内容适配 |
| `srt-translator` | 翻译字幕拓展全球受众 |

### 数据与变现
| 技能 | 功能描述 |
|:-----|:---------|

## 💡 使用示例

### 完整视频生产流水线
```
1. "找一下 [领域] 的热门话题"
2. "写一个关于 [话题] 的视频脚本"
3. "创建 5 个开场钩子变体"
4. "生成视频标题选项"
5. "创建 YouTube SEO"
6. "设计缩略图概念"
```

### 短视频工厂
```
1. "分析这个长视频的爆款片段"
2. "转换成 5 个短视频片段"
3. "为每个片段创建钩子"
4. "优化 TikTok/Shorts 标题"
```

### 国际化拓展
```
1. "获取我视频的字幕"
2. "翻译字幕到西班牙语、日语"
3. "为每个市场适配标题"
```
