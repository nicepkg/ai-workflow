# Sponsor Pitch Generator

Create compelling sponsorship proposals that convert brands into partners.

## When to Use

- Reaching out to brands for sponsorship
- Responding to brand inquiries professionally
- Creating/updating your media kit
- Negotiating sponsorship terms
- Proposing collaboration ideas

## Pitch Types

### 1. Cold Outreach Pitch
```
For: Brands you're approaching first
Tone: Professional, concise, value-focused
Length: 150-200 words max
Goal: Get a response/meeting
```

### 2. Inquiry Response
```
For: Brands who contacted you
Tone: Enthusiastic, detailed, collaborative
Length: 200-300 words
Goal: Move to negotiation
```

### 3. Media Kit
```
For: Professional overview document
Format: PDF or webpage
Content: Stats, demographics, past work, rates
Goal: Establish credibility and value
```

## Rate Calculation Guide

```
PRICING FACTORS:
â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€
1. View Count: $X per 1,000 views (CPM)
   - Industry average: $20-50 CPM
   - Tech/Finance niche: $30-100 CPM
   - Entertainment: $10-30 CPM

2. Engagement Rate: Premium for high engagement
   - 5%+ engagement: Add 20-50% premium
   - Viral potential: Add 50-100% premium

3. Production Value: Professional vs casual
   - High production: Higher rates
   - Multiple deliverables: Package pricing

4. Exclusivity: Competitor restrictions
   - Category exclusivity: Add 25-50%
   - Platform exclusivity: Add 15-25%

5. Usage Rights: Beyond YouTube
   - Social media repost: Add 25%
   - Website/ads: Add 50-100%
   - Perpetual rights: Add 100-200%

QUICK FORMULA:
Base Rate = (Average Views Ã— CPM) + Engagement Premium + Production Value

Example:
100K avg views Ã— $30 CPM = $3,000 base
+ 10% engagement premium = $3,300
+ High production (+$500) = $3,800
= Quote: $4,000 (round up)
```

## How to Use

### Generate Cold Pitch
```
Create a sponsorship pitch for:
Brand: [Company name]
Product: [What they sell]
My channel: [Description, stats]
Why it fits: [Connection to brand]
```

### Create Media Kit
```
Generate a media kit with:
Channel name: [Name]
Niche: [Topic]
Stats: [Subscribers, views, etc.]
Demographics: [Age, gender, location]
Past sponsors: [List if any]
```

### Response to Brand Inquiry
```
Help me respond to this brand inquiry:
[Paste their email]

My channel: [Brief description]
My rates: [If you have them]
```

## Sponsorship Checklist

Before sending pitch:
- [ ] Personalized to specific brand/person
- [ ] Authentic connection to product stated
- [ ] Clear value proposition
- [ ] Specific collaboration idea included
- [ ] Stats are current and accurate
- [ ] Media kit attached
- [ ] Clear CTA (meeting/call)
- [ ] Professional formatting
- [ ] Proofread for errors

Before accepting sponsorship:
- [ ] Deliverables clearly defined
- [ ] Timeline agreed upon
- [ ] Payment terms clear
- [ ] Usage rights specified
- [ ] FTC disclosure requirements understood
- [ ] Contract reviewed
