# Slidev Skill

Comprehensive guide for Slidev - a web-based presentation framework for developers.

## Overview

This skill provides complete documentation and guidance for creating, customizing, and exporting Slidev presentations. Slidev is a Markdown-based presentation tool built on Vite, Vue 3, and UnoCSS.

## Technology Stack

Slidev is built on:
- Vite - Fast build tool
- Vue 3 - Component framework
- UnoCSS - Atomic CSS engine
- Shiki - Syntax highlighter
- Monaco Editor - Code editor
- Mermaid - Diagram generation
- KaTeX - Math rendering
- VueUse - Vue utilities
- Iconify - Icon library
