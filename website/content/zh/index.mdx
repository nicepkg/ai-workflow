# AI Workflow

**你的 AI 很聪明，但每次对话都从零开始。**

每次开新对话，你都要重新解释相同的框架、最佳实践和专业标准。**一条命令改变一切。**

```bash
npx add-skill nicepkg/ai-workflow/workflows/content-creator-workflow
```

## 我们解决的痛点

| 😫 没有 AI Workflow | 🎉 有了 AI Workflow |
|:--------------------|:--------------------|
| **内容创作者**：每次都要解释 SEO 基础、H2 结构、meta 描述... | AI 已预装 SEO 最佳实践、内容框架、发布清单 |
| **营销人员**：教 UTM 参数、AIDA 文案法、漏斗优化... | AI 已配备 GTM 策略、活动模板、分析框架 |
| **视频创作者**：解释钩子、完播率心理学、封面规则... | AI 已掌握爆款模式、脚本结构、YouTube 优化 |
| **股票交易员**："MACD 意味着...、RSI 表示...、看 200 日均线..." | AI 已加载技术分析、基本面、多市场专业知识 |
| **产品经理**：反复解释 PRD 格式、用户故事、优先级... | AI 已掌握 RICE、MoSCoW、JTBD、完整产品周期 |
| **演讲者**：纠结幻灯片结构、叙事流程... | AI 构建麦肯锡 MECE 叙事，配合 Mermaid 图表 |
