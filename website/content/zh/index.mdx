# AI Workflow

**为你的 AI 编程助手赋能预置技能集**

不再需要反复教导 AI 同样的事情。一条命令，170+ 专业技能，即刻提升生产力。

## 快速开始

```bash
# 选择你的工作流，运行一条命令：
npx add-skill nicepkg/ai-workflow/workflows/content-creator-workflow

# 完成！技能已在你的 AI 助手中激活。
```

## 为什么选择 AI Workflow？

| 之前 | 之后 |
|------|------|
| "帮我写一篇博客..." 然后每次都要解释 SEO、格式、语气 | AI 已经知道你的内容策略、SEO 规则和品牌调性 |
| 手动向 AI 解释股票分析概念 | AI 即刻进行专业的技术和基本面分析 |
| 反复讲解营销框架和模板 | AI 生成符合品牌调性的文案，并带有正确的 UTM 追踪 |
| 从笔记中复制粘贴提示词 | 技能根据你的项目自动激活 |

**AI Workflow** = 让你的 AI 助手成为特定领域专家的预配置技能集。
