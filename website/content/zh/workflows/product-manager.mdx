# 产品经理工作流

为产品经理、产品负责人和产品领导者设计的 23 个专业技能集合。

## 目标用户

- 产品经理
- 产品负责人
- 产品总监 / VP
- 技术项目经理

## 安装

```bash
npx add-skill nicepkg/ai-workflow/workflows/product-manager-workflow
```

## 包含的技能

`work-intake` `user-research` `discovery-interviews-surveys` `customer-feedback-analyzer` `user-persona-creation` `competitive-analysis` `prd-generator` `writing-product-specs` `requirements` `writing-user-stories` `prioritization-effort-impact` `roadmap` `roadmap-backcast` `strategy` `stakeholders-org-design` `feature-planning` `agile` `agile-sprint-planning` `launch` `technical-launch-planner` `analytics` `forecast-premortem` `postmortem`

## 核心能力

### 需求收集与分类
- **work-intake** - 将任何请求转化为可追踪的工作项

### 发现与研究
- **user-research** - 用户访谈、调研、问题验证
- **customer-feedback-analyzer** - 多渠道反馈分析
- **user-persona-creation** - 创建详细的用户画像
- **competitive-analysis** - 市场和竞争对手分析

### 定义与规格
- **prd-generator** - 生成结构化的 PRD
- **writing-product-specs** - 带上下文和需求的功能规格
- **writing-user-stories** - 标准格式的用户故事

### 优先级与规划
- **prioritization-effort-impact** - 投入-影响 2x2 矩阵
- **roadmap** - 时间线和资源分配
- **strategy** - 产品策略和 OKR

### 交付与执行
- **feature-planning** - 实施计划
- **agile** - 冲刺计划和速度追踪

### 发布与复盘
- **launch** - 产品发布协调
- **analytics** - 产品分析和 KPI
- **postmortem** - 发布后复盘和经验总结

## 高级选项

```bash
# 全局安装
npx add-skill nicepkg/ai-workflow/workflows/product-manager-workflow --global

# 安装特定技能
npx add-skill nicepkg/ai-workflow/workflows/product-manager-workflow --skill prd-generator

# 列出所有技能
npx add-skill nicepkg/ai-workflow/workflows/product-manager-workflow --list
```
