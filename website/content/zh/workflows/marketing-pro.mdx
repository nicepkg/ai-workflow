# 营销专家工作流

你的 AI 营销团队，为数字营销人、增长黑客和产品营销经理提供 38 个预配置技能。

## 目标用户

- **数字营销人** - 多渠道营销活动
- **增长黑客** - 转化优化
- **产品营销经理** - GTM 与定位
- **需求生成专家** - 管道建设

## 安装

```bash
npx add-skill nicepkg/ai-workflow/workflows/marketing-pro-workflow
```

## 包含的技能（38 个）

### 0. 技能编排

| 技能 | 功能描述 |
|:------|:-------------|
| `skill-navigator` | 自动推荐实现目标应使用的技能 |

### 1. 受众与定位

| 技能 | 功能描述 |
|:------|:-------------|
| `marketing` | 完整营销资产套件（画像 → 文案、社交、邮件、视频脚本） |
| `buyer-persona-generator` | 创建详细的买家画像和 ICP |
| `Go-to-Market-Planner` | 完整 GTM 规划：定位、渠道、信息、发布 |
| `prd-v09-gtm-strategy` | PRD 驱动的 GTM 策略，作为可执行的发布系统 |
| `marketing-strategy-pmm` | 产品营销、定位、竞争战略卡 |

### 2. 定价与竞争

| 技能 | 功能描述 |
|:------|:-------------|
| `pricing-strategist` | SaaS 定价、层级、价值指标、实验策略 |
| `competitor-price-tracker` | 竞争对手定价监控 + 变化提醒 + 折扣周期 |
| `competitive-ads-extractor` | 提取分析竞争对手广告 |

### 3. 文案与信息

| 技能 | 功能描述 |
|:------|:-------------|
| `copy` | 专业 SaaS 文案：发现问题 → 提取利益点 → 高转化文案 |
| `ad-copy-generator` | Google、Meta、LinkedIn、TikTok 广告文案 |
| `content-optimizer` | 页面 SEO、关键词密度、meta 标签 |
| `content-brief` | SEO 内容规划简报 |

### 4. 落地页与转化

| 技能 | 功能描述 |
|:------|:-------------|
| `landing-page` | 落地页结构、CTA、社交证明、转化布局 |
| `landing-page-copywriter` | 使用 AIDA 框架的落地页文案 |
| `ui-ux-audit` | 现有页面 UX 审计，以转化为中心的建议 |

### 5. 营销活动执行

| 技能 | 功能描述 |
|:------|:-------------|
| `running-marketing-campaigns` | 计划 → 执行 → 衡量营销活动（UTM、内容、邮件、分析） |
| `utm-builder` | 统一 UTM 命名 + 批量生成 + 追踪 |
| `qr-code-generator` | 带 UTM 的批量二维码，用于线下物料 |
| `email-template-generator` | 邮件营销模板和序列 |

### 6. 触达与增长

| 技能 | 功能描述 |
|:------|:-------------|
| `cold-email-sequence-generator` | 7-14 封冷触达邮件序列 + A/B 主题行 + 跟进节奏 |
| `cold-outreach-generator` | 冷邮件/LinkedIn/跟进脚本（以会议为目标） |
| `personalization-at-scale` | 规模化个性化触达：每个潜客独特钩子 |
| `lead-research-assistant` | 识别和筛选潜在客户 |

### 7. A/B 测试与优化

| 技能 | 功能描述 |
|:------|:-------------|
| `funnel-analysis` | 漏斗瓶颈分析（转化诊断） |
| `analytics-interpretation` | GA4/GSC 数据解读 |

### 8. 发布管理

| 技能 | 功能描述 |
|:------|:-------------|
| `technical-launch-planner` | 技术产品发布：时间线、资产、指标、模板 |
| `launch-gtm-execution` | GTM 执行与销售赋能检查清单 |

### 9. 仪表盘与回顾

| 技能 | 功能描述 |
|:------|:-------------|
| `executive-dashboard-generator` | 自动结构化数据 → 趋势、异常、群组、目标追踪 |
| `product-analytics` | AARRR、留存曲线、关键指标（周回顾生成器） |
| `roi-analyzer` | ROI/NPV/LTV/CAC 敏感性分析（预算分配决策） |
| `social-media-analyzer` | 营销活动绩效与 ROI 分析 |
| `marketing-demand-acquisition` | 多渠道需求生成、付费媒体、CAC 计算器 |

### 10. SEO 与内容

| 技能 | 功能描述 |
|:------|:-------------|
| `seo-optimizer` | SEO 内容优化 |
| `serp-analysis` | SERP 分析和竞争对手情报 |
| `keyword-cluster-builder` | 关键词扩展和聚类 |

### 11. 社交媒体

| 技能 | 功能描述 |
|:------|:-------------|
| `linkedin-post-optimizer` | LinkedIn 帖子优化 |
| `social-repurposer` | 跨平台内容转换 |

## 完整营销流水线（9 阶段）

```
阶段 0：技能选择
└── skill-navigator → 自动路由到最佳技能

阶段 1：定位与受众
├── buyer-persona-generator → 创建 ICP 和画像
├── marketing → 生成完整营销资产套件
├── Go-to-Market-Planner → 完整 GTM 规划
└── prd-v09-gtm-strategy → PRD 驱动的发布策略

阶段 2：定价与竞争
├── pricing-strategist → 定义定价策略
├── competitor-price-tracker → 监控竞争对手定价
└── competitive-ads-extractor → 分析竞争对手广告

阶段 3：文案
├── copy → 所有渠道的高转化文案
├── ad-copy-generator → 所有平台的广告
└── content-brief → SEO 内容简报

阶段 4：落地页
├── landing-page → 完整 LP 结构与文案
├── landing-page-copywriter → AIDA 文案
└── ui-ux-audit → 转化优化

阶段 5：营销活动执行
├── running-marketing-campaigns → 完整活动规划
├── utm-builder → 统一追踪
├── qr-code-generator → 线下物料追踪
└── email-template-generator → 邮件序列

阶段 6：触达与增长
├── cold-email-sequence-generator → 冷触达序列
├── cold-outreach-generator → 以会议为目标的脚本
├── personalization-at-scale → 规模化个性化
└── lead-research-assistant → 潜客筛选

阶段 7：测试与优化
├── funnel-analysis → 诊断转化瓶颈
└── analytics-interpretation → 数据解读

阶段 8：发布
├── technical-launch-planner → 产品发布规划
└── launch-gtm-execution → 执行检查清单

阶段 9：仪表盘与回顾
├── executive-dashboard-generator → 高管报告
├── product-analytics → AARRR 指标回顾
├── roi-analyzer → ROI/LTV/CAC 分析
└── social-media-analyzer → 活动 ROI
```

## 示例工作流

### 完整 GTM 发布

```
1. "为 [产品] 创建 ICP 和 3 个买家画像"
2. "使用 Go-to-Market-Planner 开发定位"
3. "创建 3 层定价策略"
4. "生成完整营销资产套件"
5. "规划技术产品发布"
6. "创建带转化优化的落地页"
7. "为所有渠道设置 UTM 追踪"
```

### 全漏斗营销活动

```
1. "为 [产品] 规划需求生成活动"
2. "为 Google 和 LinkedIn 生成广告文案"
3. "创建 7 封冷触达邮件序列"
4. "为 100 个潜客进行个性化触达"
5. "为所有渠道创建 UTM 链接"
6. "创建落地页文案"
7. "分析漏斗瓶颈"
```

### 增长触达机器

```
1. "为 [ICP] 研究和筛选 50 个潜客"
2. "为每个潜客创建个性化触达"
3. "生成带 A/B 主题行的冷邮件序列"
4. "设置 LinkedIn 触达脚本"
5. "追踪活动绩效"
```

### 产品发布

```
1. "创建 GTM 策略文档"
2. "规划技术发布时间线"
3. "生成发布资产（落地页、邮件、社交）"
4. "创建追踪用高管仪表盘"
5. "执行 GTM 检查清单"
```

## 技能组合

| 目标 | 技能链 |
|:-----|:------------|
| **产品发布** | buyer-persona → Go-to-Market-Planner → pricing-strategist → marketing → technical-launch-planner |
| **冷触达** | lead-research → personalization-at-scale → cold-email-sequence-generator → analytics |
| **落地页** | copy → landing-page → ui-ux-audit → utm-builder |
| **A/B 测试** | running-marketing-campaigns → funnel-analysis → roi-analyzer |
| **SEO 活动** | keyword-cluster → serp-analysis → content-brief → content-optimizer |
| **付费媒体** | competitive-ads → ad-copy → utm-builder → social-media-analyzer |
| **周回顾** | executive-dashboard-generator → product-analytics → roi-analyzer |
