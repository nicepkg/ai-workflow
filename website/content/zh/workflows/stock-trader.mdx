# 股票交易员工作流

为金融分析师、交易员和投资研究员提供的 29 个专业技能集合，覆盖美股、A股、港股和台股市场。

## 目标用户

- 股票交易员
- 金融分析师
- 投资研究员
- 投资组合经理
- 量化分析师

## 安装

```bash
npx add-skill nicepkg/ai-workflow/workflows/stock-trader-workflow
```

## 包含的技能

`akshare` `股票分析` `stock-screener` `shioaji` `us-stock-analysis` `a-share-analysis` `hk-stock-analysis` `technical-analyst` `china-macro-analyst` `cross-border-flow-tracker` `a-share-screener` `portfolio-manager` `earnings-calendar` `market-news-analyst` `canslim-screener` `value-dividend-screener` `sector-analyst` `options-strategy-advisor` `backtest-expert` `scenario-analyzer` `institutional-flow-tracker` `market-environment-analysis` `breadth-chart-analyst` `pair-trade-screener` `dividend-growth-pullback-screener` `economic-calendar-fetcher` `weekly-trade-strategy` `us-market-bubble-detector` `stanley-druckenmiller-investment`

## 核心能力

- **市场分析** - 美股、A股、港股分析
- **技术分析** - 图表形态、技术指标、广度分析
- **基本面分析** - CANSLIM、价值投资、股息筛选
- **投资组合管理** - 资产配置、回测、情景分析
- **市场情报** - 新闻分析、机构资金流向、宏观分析

## 支持的市场

- **美股市场** - NYSE、NASDAQ
- **中国 A 股** - 上海、深圳
- **香港** - 港交所
- **台湾** - 台湾证券交易所

## 免责声明

本工作流仅提供研究和分析工具，不构成投资建议。在做出投资决策前，请务必咨询合格的金融专业人士。

## 高级选项

```bash
# 全局安装
npx add-skill nicepkg/ai-workflow/workflows/stock-trader-workflow --global

# 安装特定技能
npx add-skill nicepkg/ai-workflow/workflows/stock-trader-workflow --skill a-share-analysis

# 列出所有技能
npx add-skill nicepkg/ai-workflow/workflows/stock-trader-workflow --list
```
