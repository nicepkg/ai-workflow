# 工作流

AI Workflow 为不同专业需求提供精选的技能集合。每个工作流都包含针对特定用例优化的精心挑选的技能。

## 可用工作流

| 工作流                                         | 适用人群                             | 技能数 | 安装命令                                                               |
| :--------------------------------------------- | :----------------------------------- | :----: | :--------------------------------------------------------------------- |
| [内容创作者](/zh/workflows/content-creator)    | 博主、作家、SEO 专家                 |   32   | `npx add-skill nicepkg/ai-workflow/workflows/content-creator-workflow` |
| [营销专家](/zh/workflows/marketing-pro)        | 数字营销人、增长黑客、PMM            |   38   | `npx add-skill nicepkg/ai-workflow/workflows/marketing-pro-workflow`   |
| [视频创作者](/zh/workflows/video-creator)      | YouTuber、TikToker、视频制作人       |   29   | `npx add-skill nicepkg/ai-workflow/workflows/video-creator-workflow`   |
| [股票交易员](/zh/workflows/stock-trader)       | 交易员、投资者（美股/A股/港股/台股） |   29   | `npx add-skill nicepkg/ai-workflow/workflows/stock-trader-workflow`    |
| [产品经理](/zh/workflows/product-manager)      | PM、产品负责人、产品总监             |   23   | `npx add-skill nicepkg/ai-workflow/workflows/product-manager-workflow` |
| [Talk to Slidev](/zh/workflows/talk-to-slidev) | 演讲者、技术讲师、教育工作者         |   18   | `npx add-skill nicepkg/ai-workflow/workflows/talk-to-slidev-workflow`  |

## 快速安装

```bash
# 选择你的工作流，运行一条命令：
npx add-skill nicepkg/ai-workflow/workflows/content-creator-workflow

# 完成！技能现在已在你的 AI 助手中激活。
```

## 高级选项

```bash
# 全局安装（在所有项目中可用）
npx add-skill <workflow> --global

# 只安装特定技能
npx add-skill <workflow> --skill skill-name

# 列出可用技能
npx add-skill <workflow> --list
```

## 创建自定义工作流

需要针对你特定用例的工作流？请参阅 [创建工作流](/zh/docs/creating-workflows) 来构建你自己的工作流。
