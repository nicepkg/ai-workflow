# Talk to Slidev 工作流

为主题演讲者、技术大会讲者和教育工作者设计的 18 个专业技能集合，用于创建精美的 Slidev 演示文稿。

## 目标用户

- 主题演讲者
- 技术大会讲者
- 创业者（路演 Pitch Deck）
- 教育工作者和培训师

## 安装

```bash
npx add-skill nicepkg/ai-workflow/workflows/talk-to-slidev-workflow
```

## 包含的技能

`storyline-builder` `presentation-builder` `ppt-creator` `presentation-generator` `theme-factory` `presentation-design-enhancer` `color-palette-extractor` `font-pairing-suggester` `canvas-design` `pexels-media` `nano-banana-pro` `youtube-transcript` `transcribe-and-analyze` `mermaid-diagrams` `mermaid-tools` `infographic-creation` `pptx` `social-repurposer`

## 核心能力

### 创意到叙事结构
- **storyline-builder** - 麦肯锡 MECE 叙事框架
- **presentation-builder** - 通用演示结构

### 脚本与演讲备注
- **ppt-creator** - 完整的演讲稿和演讲备注
- **presentation-generator** - HTML 演示文稿生成

### 视觉主题与设计
- **theme-factory** - 一键主题选择
- **presentation-design-enhancer** - 将要点转化为视觉效果
- **color-palette-extractor** - 从图片提取配色方案
- **font-pairing-suggester** - 专业字体搭配

### 媒体与素材
- **pexels-media** - 查找免版权图片和视频
- **nano-banana-pro** - AI 生成插图
- **youtube-transcript** - 从 YouTube 提取内容

### 图表与信息图
- **mermaid-diagrams** - 流程图、架构图
- **mermaid-tools** - 高级 Mermaid 编辑
- **infographic-creation** - 将文本转化为信息图

### 导出与复用
- **pptx** - 导出为 PowerPoint 格式
- **social-repurposer** - 将幻灯片复用到社交媒体

## 高级选项

```bash
# 全局安装
npx add-skill nicepkg/ai-workflow/workflows/talk-to-slidev-workflow --global

# 安装特定技能
npx add-skill nicepkg/ai-workflow/workflows/talk-to-slidev-workflow --skill ppt-creator

# 列出所有技能
npx add-skill nicepkg/ai-workflow/workflows/talk-to-slidev-workflow --list
```
