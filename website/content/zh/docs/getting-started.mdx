# 快速开始

AI Workflow 为 AI 编程助手提供预配置的技能集。本指南将帮助你快速上手。

## 前置条件

- 已安装 Node.js 18+
- 一个 AI 编程助手（Claude Code、Cursor、Copilot 等）

## 快速开始

### 安装完整工作流（推荐）

```bash
# 选择你的工作流，运行一条命令：
npx add-skill nicepkg/ai-workflow/workflows/content-creator-workflow

# 完成！技能已在你的 AI 助手中激活。
```

### 可用工作流

| 工作流         | 适用人群                       | 技能数 | 安装命令                                                               |
| :------------- | :----------------------------- | :----: | :--------------------------------------------------------------------- |
| 内容创作者     | 博主、写手、SEO 专家           |   32   | `npx add-skill nicepkg/ai-workflow/workflows/content-creator-workflow` |
| 营销专家       | 数字营销人、增长黑客、PMM      |   38   | `npx add-skill nicepkg/ai-workflow/workflows/marketing-pro-workflow`   |
| 视频创作者     | YouTuber、TikToker、视频制作人 |   29   | `npx add-skill nicepkg/ai-workflow/workflows/video-creator-workflow`   |
| 股票交易员     | 交易员、投资者（美/中/港/台）  |   29   | `npx add-skill nicepkg/ai-workflow/workflows/stock-trader-workflow`    |
| 产品经理       | PM、产品负责人                 |   23   | `npx add-skill nicepkg/ai-workflow/workflows/product-manager-workflow` |
| Talk to Slidev | 演讲者、讲师、教育工作者       |   18   | `npx add-skill nicepkg/ai-workflow/workflows/talk-to-slidev-workflow`  |

### 安装单个技能

```bash
# 只安装你需要的
npx add-skill nicepkg/ai-workflow/workflows/stock-trader-workflow --skill a-share-analysis

# 先列出可用技能
npx add-skill nicepkg/ai-workflow/workflows/marketing-pro-workflow --list
```

### 高级选项

```bash
# 全局安装（在所有项目中可用）
npx add-skill nicepkg/ai-workflow/workflows/video-creator-workflow --global

# 只安装到指定的 AI 工具
npx add-skill nicepkg/ai-workflow/workflows/content-creator-workflow -a claude-code -a cursor

# 非交互模式（用于 CI/CD）
npx add-skill nicepkg/ai-workflow/workflows/marketing-pro-workflow -y
```

## 验证安装

安装后，技能应该会自动在你的 AI 助手中可用。尝试在对话中提及技能名称或用途来使用它。
