# Talk to Slidev Workflow

A specialized collection of 18 skills for keynote speakers, tech conference presenters, and educators to create beautiful Slidev presentations.

## Target Users

- Keynote Speakers
- Tech Conference Presenters
- Startup Founders (Pitch Decks)
- Educators & Trainers

## Installation

```bash
npx add-skill nicepkg/ai-workflow/workflows/talk-to-slidev-workflow
```

## Included Skills

`storyline-builder` `presentation-builder` `ppt-creator` `presentation-generator` `theme-factory` `presentation-design-enhancer` `color-palette-extractor` `font-pairing-suggester` `canvas-design` `pexels-media` `nano-banana-pro` `youtube-transcript` `transcribe-and-analyze` `mermaid-diagrams` `mermaid-tools` `infographic-creation` `pptx` `social-repurposer`

## Key Capabilities

### Idea to Narrative Structure
- **storyline-builder** - McKinsey MECE narrative framework
- **presentation-builder** - General presentation structure

### Script & Speaker Notes
- **ppt-creator** - Complete speech script and speaker notes
- **presentation-generator** - HTML deck generation

### Visual Theme & Design
- **theme-factory** - One-click theme selection
- **presentation-design-enhancer** - Transform bullets into visuals
- **color-palette-extractor** - Extract color schemes from images
- **font-pairing-suggester** - Professional font combinations

### Media & Assets
- **pexels-media** - Find royalty-free images and videos
- **nano-banana-pro** - AI-generated illustrations
- **youtube-transcript** - Extract content from YouTube

### Diagrams & Infographics
- **mermaid-diagrams** - Flowcharts, architecture diagrams
- **mermaid-tools** - Advanced Mermaid editing
- **infographic-creation** - Transform text into infographics

### Export & Repurpose
- **pptx** - Export to PowerPoint format
- **social-repurposer** - Repurpose slides for social media

## Advanced Options

```bash
# Install globally
npx add-skill nicepkg/ai-workflow/workflows/talk-to-slidev-workflow --global

# Install specific skill
npx add-skill nicepkg/ai-workflow/workflows/talk-to-slidev-workflow --skill ppt-creator

# List all skills
npx add-skill nicepkg/ai-workflow/workflows/talk-to-slidev-workflow --list
```
