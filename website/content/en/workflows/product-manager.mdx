# Product Manager Workflow

A professional collection of 23 skills designed for product managers, product owners, and product leaders.

## Target Users

- Product Managers
- Product Owners
- Head of Product / VP
- Technical Program Managers

## Installation

```bash
npx add-skill nicepkg/ai-workflow/workflows/product-manager-workflow
```

## Included Skills

`work-intake` `user-research` `discovery-interviews-surveys` `customer-feedback-analyzer` `user-persona-creation` `competitive-analysis` `prd-generator` `writing-product-specs` `requirements` `writing-user-stories` `prioritization-effort-impact` `roadmap` `roadmap-backcast` `strategy` `stakeholders-org-design` `feature-planning` `agile` `agile-sprint-planning` `launch` `technical-launch-planner` `analytics` `forecast-premortem` `postmortem`

## Key Capabilities

### Intake & Triage
- **work-intake** - Convert any request into trackable work items

### Discovery & Research
- **user-research** - User interviews, surveys, problem validation
- **customer-feedback-analyzer** - Multi-channel feedback analysis
- **user-persona-creation** - Create detailed user personas
- **competitive-analysis** - Market and competitor analysis

### Definition & Specification
- **prd-generator** - Generate structured PRDs
- **writing-product-specs** - Feature specs with context and requirements
- **writing-user-stories** - Standard format user stories

### Prioritization & Planning
- **prioritization-effort-impact** - Effort-Impact 2x2 matrix
- **roadmap** - Timeline and resource allocation
- **strategy** - Product strategy and OKRs

### Delivery & Execution
- **feature-planning** - Implementation plans
- **agile** - Sprint planning and velocity tracking

### Launch & Review
- **launch** - Product launch coordination
- **analytics** - Product analytics and KPIs
- **postmortem** - Post-launch review and lessons learned

## Advanced Options

```bash
# Install globally
npx add-skill nicepkg/ai-workflow/workflows/product-manager-workflow --global

# Install specific skill
npx add-skill nicepkg/ai-workflow/workflows/product-manager-workflow --skill prd-generator

# List all skills
npx add-skill nicepkg/ai-workflow/workflows/product-manager-workflow --list
```
