# Content Creator Workflow

A curated collection of 32 skills designed for content creators, bloggers, and writers.

## Target Users

- Bloggers and article writers
- Technical documentation authors
- SEO specialists
- Newsletter publishers
- Social media content creators

## Installation

```bash
npx add-skill nicepkg/ai-workflow/workflows/content-creator-workflow
```

## Included Skills

`weak-signal-synthesizer` `content-trend-researcher` `competitive-ads-extractor` `twitter-reader` `article-extractor` `youtube-processor` `youtube-transcript` `content-research` `content-brief-generator` `content-brief` `skill-navigator` `blog-post-writer` `content-research-writer` `newsletter-coach` `fact-checker` `hook-stack-evaluator` `email-subject-line-optimizer` `ai-slop-detector` `content-repurposer` `social-repurposer` `social-media` `linkedin-announcement-generator` `podcast-content-suite` `pre-publish-post-assistant` `seo-wordpress-manager` `gsc-assistant` `astro-cta-injector` `nano-banana` `canvas-design` `docx` `pdf`

## Key Capabilities

- **Content Research** - Trend analysis, competitive research, topic ideation
- **Writing Assistance** - Blog posts, newsletters, social media content
- **SEO Optimization** - WordPress SEO, Google Search Console integration
- **Content Repurposing** - Transform content across platforms
- **Quality Assurance** - Fact checking, AI content detection

## Advanced Options

```bash
# Install globally
npx add-skill nicepkg/ai-workflow/workflows/content-creator-workflow --global

# Install specific skill
npx add-skill nicepkg/ai-workflow/workflows/content-creator-workflow --skill blog-post-writer

# List all skills
npx add-skill nicepkg/ai-workflow/workflows/content-creator-workflow --list
```
